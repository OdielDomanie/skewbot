# skewbot

Discord bot that *italicizes* images, by creating a gif animation of the image getting skewed.

Invite link to the canonical instance of bot: https://discord.com/api/oauth2/authorize?client_id=973905136562864150&permissions=0&scope=bot%20applications.commands


## Installation

To host the bot yourself, do inside a virtual environment:

`pip install --upgrade git+https://github.com/OdielDomanie/skewbot.git`

Then configure the environment variables (possibly in a .env file), and do:

`python -m skewbot`

To unregister the command, do:

`python -m skewbot clear`
